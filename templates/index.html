F E N I X
VUELOS RESERVA PAGOS

La Aerolínea “FENIX” realiza vuelos internacionales y nacionales y tiene el agrado de mostrar su gestion de base de datos donde se podra realizar lo sigueinte:


Permitir registrar, mostrar, actualizar y borrar los diferentes vuelos que hace la aerolínea.


Permitir realizar reservas para los pasajeros (CRUD).


Realizar pagos de las reservas que hicieron los pasajeros
